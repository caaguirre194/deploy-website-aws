# Deploying a Website to AWS with Terraform
\
![GitHub](/img/architecture.svg)

## Config
- Add your AWS credentials in the following variables within variables.tf

	- user_information 
	- layer 
	- aliases 
	- domain_name 


## Deploying

```bash
$ terraform init
$ terraform apply
```
